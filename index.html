About

StarcraftLab handles UMS replays and stores win/loss records automatically. The player just uploads his replays and he gets all the results organised.
Implementation by Massacre^2
BattleNet Tag: KKC99#2153

Ongoing Project
  • 1st project is Desert Strike Classic 1.9
  • Max players 4 Games: 1v1, 1v2, 2v2
  • Win/Loss Script Yes
  • Upload .rep files with 1 button Yes
  • Visual Graphs/Rankings/Statistics Yes
Implementation

End-to-end Python analytics pipeline for game replay data with automated parsing, KPIs, and visual dashboards using Pandas, NumPy, Matplotlib, and Plotly; integrated with a PostgreSQL database and a public web interface for user uploads.
